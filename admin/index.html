ℹ️ Cum funcționează

Imaginile sunt încărcate direct în repository-ul GitHub via API. Recomandare: format JPG/WebP, dimensiune 1200×630px, max 500KB. Site-ul se actualizează automat în 1-2 minute după publicare.

ℹ️ Editare conținut programe

Modificările sunt salvate direct în fișierele HTML via GitHub API și devin publice în 1-2 minute. Câmpurile editate sunt: sumă maximă, procent nerambursabil, termen limită, status badge și deadline countdown.
