🚀 Pași pentru indexare Google
1. Mergi la Google Search Console și adaugă atelierdeconsultanta.ro
2. Alege verificarea prin HTML tag și copiază codul de verificare
3. Mergi la SEO & Meta tags → câmpul "Google Verification" → lipește codul → Salvează
4. Întoarce-te în Search Console și apasă Verifică
5. În Search Console → Sitemaps → adaugă https://atelierdeconsultanta.ro/sitemap.xml
6. Apasă Indexare URL pentru pagina principală → Solicită indexare
✅ robots.txt și sitemap.xml sunt deja pe site. Indexarea completă durează 1-4 săptămâni.
