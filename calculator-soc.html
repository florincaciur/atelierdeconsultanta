Calculator Standard Output (SO)

Calculează dimensiunea economică a fermei tale și verifică dacă îndeplinești pragurile de eligibilitate pentru programele AFIR DR12 (Tineri Fermieri) și DR14 (Ferme Mici). Instrumente gratuite pentru antreprenorii din agricultură.

Informații esențiale

Ce este Standard Output (SO)?

Standard Output (SO) reprezintă producția standard pe unitate (hectar pentru culturi, cap de animal pentru zootehnie) exprimată în EUR. Se calculează pe baza datelor PAC 2023-2027 pentru România și determină dimensiunea economică a fermei. Aceasta este esențială pentru eligibilitatea la programele AFIR.

Clasificare Fermă pe Bază SO

Verifică pe care dintre categoriile de mai jos te-ai încadra în funcție de SO-ul calculat

🌾
DR14

Ferme Mici

4.000 – 11.999 SO

Eligibile pentru Submasura DR14 – Investiții în ferme mici. Eligibilitate: minimum 4.000 SO și maximum 11.999 SO pe o singură cultură/zootehnie sau combinații.

👨‍🌾
DR12

DR12 Tineri Fermieri

12.000 – 100.000 SO

Eligibili pentru Submasura DR12 – Sprijin pentru tineri fermieri. Cerință: minimum 12.000 SO și maximum 100.000 SO. Vârsta beneficiarului: sub 40 ani.

🏭
Fermă Comercială

Ferme Comerciale

> 100.000 SO

Ferme cu SO peste 100.000 EUR. Se clasifică ca ferme comerciale mari, cu alte opțiuni de finanțare și obligații specifice diferite de programele AFIR DR12/DR14.
